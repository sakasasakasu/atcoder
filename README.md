# WIP
AtCoderの精進をする過程で気づいたことや反省をまとめています。

今まではNotionでまとめていたのですが、エンジニアっぽさがないのでGithubを使う方針でやろうと思います。

いずれは様々な方法でまとめたいという思いがあります。

## 技術スタック
Next.jsを採用。ReactはUIコンポーネントライブラリが充実しているので簡単に作れるかなと思って、こうしました。

案外そんなことはなくて、GUIのセンスを問われるという感じでしたね。

一応mainブランチにpushされるとGithub ActionsABCディレクトリ以降の情報をパースして内容をjsonに変換するようにしています。

同時にサイトをビルドして静的サイトとしてGithub pagesに公開されます！
